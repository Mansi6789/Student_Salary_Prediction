# Student Salary Prediction App

This project predicts the **expected salary package (LPA)** of students based on their academic scores, coding skills, internships, aptitude, and mock interview performance using Machine Learning.

---

## **Features**

- Predict salary using **Random Forest** or **Linear Regression** models
- Interactive **Streamlit app** for real-time prediction
- **Feature importance visualization** to understand which factors impact salary
- Actual vs Predicted salary comparison plot
- Optionally, download the prediction report

---

## **Technologies Used**

- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (Linear Regression, Random Forest)
- Joblib
- Streamlit
